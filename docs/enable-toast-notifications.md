# メッセージ受信時にトースト通知を表示する方法

このリポジトリは Mattermost の **Docker デプロイ用** なので、
「メッセージ受信時にトーストを出す」挙動自体はアプリ（Web/デスクトップクライアント）側の設定で有効化します。

## 1. ブラウザ通知を許可する

1. Mattermost をブラウザで開く
2. アドレスバーのサイト権限で「通知」を **許可** にする

## 2. Mattermost の通知設定を有効化する

1. 右上のプロフィールメニューを開く
2. **Preferences（設定）** → **Notifications（通知）**
3. **Desktop notifications（デスクトップ通知）** を有効化
4. 通知対象（すべて / メンションのみ など）を選ぶ

## 3. （管理者）通知機能が無効化されていないか確認する

管理者がサーバー設定で通知関連機能を制限している場合、ユーザー側で有効化できません。
必要に応じて System Console の通知関連設定を確認してください。

---

補足: Docker 構成側で追加のコンテナ設定を行わなくても、通常は上記でトースト通知を表示できます。
